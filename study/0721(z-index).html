상자의 테두리를 둥글게 만드는 속성 border-radius: 5px; (반지름이 5px인 사분원을 꼭짓점에 그린다) z-index 속성 z축은 모니터와 사람이 바라보는 눈의 축이며, z-index를 설정하면 특정 요소를 앞으로 가져오거나 뒤로 보낼 수 있다. z-index는 position이 static일 때 사용해도 효과가 없다. position이 static(기본값)인 경우 순서대로 쌓이게 된다. position이 relative인 경우, static 속성을 갖고 있는 요소보다 z-index가 높게 설정된다. 따라서 static 속성이 아닌 요소는 무조건 static 요소보다 위에 쌓인다. position이 static이 아닌 요소들 끼리는 나중 요소가 위에 쌓인다(순서대로 쌓인다). z-index는 간단한 요소끼리의 상하관계를 설정할 때만 사용한다. media query 가로길이, 세로길이 혹은 html 문서를 열고있는 기기의 종류
